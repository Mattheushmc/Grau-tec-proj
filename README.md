# PROJETO TABUADA

A iniciativa do Projeto(Tabuada) é fazer uma simples consulta das quatros operações:

- Adição

- Subtração

- Multiplicação

- Divisão

O projeto também abrange outros algoritmos como os citados abaixo:

- Área de um Retângulo

- Fatorial

- Fibonnaci 

- Imposto

- Maior

- Média

- Primos

- Crescente-Decrescente

- Par
